org.hammerlab.guacamole.jointcaller

Input

case class Input(index: SampleId, sampleName: SampleName, path: String, tissueType: Input.TissueType.Value, analyte: Input.Analyte.Value) extends Product with Serializable

An input BAM to the joint variant caller.

The caller can work with any number of normal and tumor BAMs, each of which may be DNA or RNA.

index

Throughout the joint caller, we refer to inputs by index, which is the index (0 <= index < num inputs) of the input as specified on the commandline

sampleName

arbitrary name for this sample, used in VCF output

path

path to BAM

tissueType

tumor or normal

analyte

rna or dna
